# Manuel interactif : Tectonique des plaques

Ce dépôt contient un manuel interactif pour réviser la tectonique des plaques au collège / lycée.  
Il est illustré, structuré en chapitres et se termine par un quiz autocorrigé.

## Accès en ligne (recommandé)

La version la plus simple à utiliser est hébergée sur GitHub Pages :

- **Manuel en ligne** : https://66julienmartin.github.io/cours_tectonique_plaques/

Ouvrez simplement ce lien depuis :

- un ordinateur (navigateur web classique) ;
- une tablette ou un smartphone (Chrome, Safari, etc.).

Tout le contenu (texte, images et quiz) est alors chargé automatiquement.

## Structure du manuel

Le fichier principal est :

- `index.html` – le manuel complet :
  - Chapitre 1 : L'enquête (preuves de la dérive des continents) ;
  - Chapitre 2 : La structure interne (lithosphère / asthénosphère) ;
  - Chapitre 3 : Le moteur (chaleur interne et convection) ;
  - Chapitre 4 : L'action (divergence, subduction, collision) ;
  - Chapitre 5 : Les risques et la protection ;
  - Résumé final + quiz interactif (10 questions).

Les fichiers d'images associés (schémas et illustrations) sont :

- `unnamed1.jpg` à `unnamed7.jpg`

Ils sont déjà intégrés dans la page et s'affichent automatiquement tant qu'ils restent au même niveau que `index.html`.

## Utilisation en local (hors ligne)

Si vous souhaitez utiliser le manuel sans connexion Internet :

1. Téléchargez le dépôt (bouton **Code** → **Download ZIP** sur GitHub).
2. Décompressez le ZIP sur votre ordinateur ou tablette.
3. Ouvrez `index.html` dans un navigateur web :
   - sur ordinateur : double‑clic sur `index.html` ;
   - sur tablette/smartphone : ouvrir le fichier depuis l'application de gestion de fichiers.

Le manuel fonctionne entièrement en local, y compris les images et le quiz.

## À propos du quiz

- 10 questions couvrent les notions clés du manuel.
- Le bouton **« Vérifier mes réponses »** donne un score et un retour détaillé sans dévoiler directement les bonnes réponses.
- Le corrigé complet est disponible dans un bloc repliable **« Afficher / masquer le corrigé »** à la fin du quiz, pour une correction guidée.

Ce support peut être utilisé :

- en révision individuelle par les élèves ;
- en activité projetée en classe (vidéoprojecteur / TNI) ;
- comme base pour créer d'autres quiz ou chapitres sur la géologie.
